## Local development

To run this project locally, clone the repository and install the dependencies:

\`\`\`bash
pnpm install
\`\`\`

Then, run the development server:

\`\`\`bash
pnpm dev
\`\`\`

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
